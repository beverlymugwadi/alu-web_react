alu-web_react

Small collection of React and Webpack practice tasks.

Project layout
- `react_intro/`
  - `task_0/` … `task_5/` — each task in its own folder

Quick commands

- Task 3 (Create React App + tests)
  ```bash
  cd react_intro/task_3/dashboard
  npm install
  npm test
  ```

- Task 4 (Deploy to GitHub Pages)
  ```bash
  cd react_intro/task_4/dashboard
  # set "homepage" in package.json: https://<USERNAME>.github.io/alu-web_react
  npm install
  npm run deploy
  ```

- Task 5 (Webpack project)
  ```bash
  cd react_intro/task_5/dashboard
  npm install
  npm start   
  ```
